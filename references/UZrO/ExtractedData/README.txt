Each of the folders here contain the data extracted, renamed or otherwise from the LAMMPS output files for direct use in MATLAB scripts.

LatticeParameter - contains the data used to calculate lattice parameter, thermal expansion and heat capacity
LiteratureData - contains .txt files which have the experimental data points used for plotting
MeltingPoint - contains a txt file which has the melting temperatures and compositions in it
Plots - Contains all exported Plots
RadialDistributionFunction - Contains data for the radial distribution function plots, g(r)
